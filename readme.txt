WP Monitor.us displays information from your [Monitor.us](http://monitor.us) account on the WordPress dashboard.

== Description ==

WP Monitor.us connects to your Monitor.us account and displays information from your monitors right on the WordPress dashboard. Get a quick overview of the status your sites right in one spot.

This plugin requires PHP >= 5.3.0.

Major features include:

* A WP Dashboard widget that provides the latest snapshot of your monitors from numerous locations around the world.
* A WP Dashboard widget that provides summary information for the last 24 hours of all configured monitors.

Please note: This plugin is in no way associtated with Monitor.us or Monitis. This is a third party plugin.

== Installation ==

1. Upload `wp-monitorus.zip` to the `/wp-content/plugins/` directory and extract.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the settings page( Settings >> WP Monitor.us ), and retrieve your Monitor.us API key.
